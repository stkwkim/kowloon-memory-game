# 🏙️ 九龍記憶庫 - AR歷史解謎遊戲

## 項目簡介
「九龍記憶庫」是一個結合AR技術的歷史解謎遊戲，讓參與者透過智能手機探索土瓜灣至九龍城的百年歷史。項目旨在推廣社區歷史教育，同時收集參與者數據用於社區建設。

## 功能特色
- 🎮 沉浸式AR歷史體驗
- 💰 多層次付費方案
- 📊 大數據收集與分析
- 🤝 義工組織對接
- 📱 響應式網頁設計
- 🎯 社交分享功能

## 價格方案
